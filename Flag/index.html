A fun fact about the U.S. flag is that the current design was created by a 17-year-old high school student, Robert Heft, for a school project in 1958 and was later chosen by President Dwight D. Eisenhower. The flag has also been officially updated 27 times since it was first adopted, with each new version adding a star for a new state joining the Union.
